# 企业微信自动化 v1.2 设计文档

## 新增功能：批量测试与结果分类

### 需求
- 批量测试多个手机号
- 存在的手机号正常添加
- 不存在的手机号记录到新数组
- 返回分类统计结果

### 实现

#### 1. 批量测试函数

```python
def test_phones_batch(self, phones, verify_msg=None):
    """批量测试手机号，分类结果
    
    Args:
        phones: 手机号列表
        verify_msg: 验证消息
        
    Returns:
        dict: {
            "exists": [],      # 存在的手机号
            "not_exist": [],   # 不存在的手机号
            "already_friend": [],  # 已是好友
            "failed": [],      # 其他失败
            "summary": {}      # 统计信息
        }
    """
```

#### 2. 结果分类

| 状态 | 处理方式 |
|------|---------|
| `success` | 添加到 `exists` 列表 |
| `phone_not_exist` | 添加到 `not_exist` 列表 |
| `already_friend` | 添加到 `already_friend` 列表 |
| `failed`/`not_found` | 添加到 `failed` 列表 |

#### 3. 使用示例

```python
wx = WXWorkAutomation()
phones = [
    "16670239176",
    "13611111111",
    "13611111122",
    "13682416565"
]

result = wx.test_phones_batch(phones, verify_msg="你好")

print("存在的手机号:", result["exists"])
print("不存在的手机号:", result["not_exist"])
print("已是好友:", result["already_friend"])
print("失败的:", result["failed"])

# 保存不存在的手机号到文件
with open("not_exist_phones.txt", "w") as f:
    for phone in result["not_exist"]:
        f.write(phone + "\n")
```

#### 4. 增强功能

- 进度显示：处理第 N/M 个
- 间隔控制：避免操作过快
- 错误重试：失败时重试一次
- 结果导出：保存到 JSON/TXT

## 实现代码

```python
def test_phones_batch(self, phones, verify_msg=None, interval=3):
    """批量测试手机号"""
    results = {
        "exists": [],
        "not_exist": [],
        "already_friend": [],
        "failed": [],
        "details": []
    }
    
    total = len(phones)
    for i, phone in enumerate(phones, 1):
        logger.info(f"[{i}/{total}] 测试手机号: {phone}")
        
        # 执行添加
        result = self.add_contact_by_phone(phone, verify_msg)
        result["phone"] = phone
        result["index"] = i
        
        # 分类结果
        status = result["status"]
        if status == "success":
            results["exists"].append(phone)
            logger.info(f"  ✓ 存在，添加成功")
        elif status == "phone_not_exist":
            results["not_exist"].append(phone)
            logger.info(f"  ✗ 不存在")
        elif status == "already_friend":
            results["already_friend"].append(phone)
            logger.info(f"  ℹ️ 已是好友")
        else:
            results["failed"].append(phone)
            logger.warning(f"  ✗ 失败: {result['detail']}")
        
        results["details"].append(result)
        
        # 间隔
        if i < total and interval > 0:
            time.sleep(interval)
    
    # 统计
    results["summary"] = {
        "total": total,
        "exists": len(results["exists"]),
        "not_exist": len(results["not_exist"]),
        "already_friend": len(results["already_friend"]),
        "failed": len(results["failed"])
    }
    
    return results
```
