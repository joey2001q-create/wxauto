# 企业微信自动化 v1.1 设计文档

## 新增功能：手机号不存在检测

### 问题描述
当前版本在搜索手机号时，如果该手机号不存在或无法添加，界面不会出现"网络查找"结果，但代码无法区分这种情况，只能返回"未找到搜索结果"。

### 解决方案

#### 1. 检测"无搜索结果"提示

企业微信在搜索不到结果时，会显示特定提示文字，如：
- "无搜索结果"
- "该用户不存在"
- "暂无搜索结果"

#### 2. 状态扩展

新增状态 `phone_not_exist`：

```python
{
    "status": "phone_not_exist",  # 手机号不存在
    "detail": "该手机号未找到对应用户"
}
```

#### 3. 完整状态列表

| 状态 | 说明 |
|------|------|
| `success` | 添加邀请发送成功 |
| `already_friend` | 已是好友 |
| `not_found` | 未找到搜索结果（网络问题/OCR问题） |
| `phone_not_exist` | **新增**：手机号不存在 |
| `failed` | 其他失败原因 |

#### 4. 实现逻辑

```python
def add_contact_by_phone(self, phone, verify_msg=None):
    # ... 输入手机号后 ...
    
    # 等待搜索结果出现
    time.sleep(2)
    
    # 检查是否出现"无搜索结果"提示
    no_result_hints = ["无搜索结果", "该用户不存在", "暂无搜索结果", "未找到"]
    for hint in no_result_hints:
        if self.find_text(hint):
            self.press_escape()
            return {
                "status": "phone_not_exist",
                "detail": f"手机号 {phone} 不存在或无法添加"
            }
    
    # 原有逻辑：查找"网络查找"
    lookup_item = self.find_text_safe("网络查找")
    if not lookup_item:
        # 再次检查是否真的没有结果
        # 可能是OCR漏识别，截图保存用于调试
        self.screenshot(save_path=f"debug_no_result_{phone}.png")
        self.press_escape()
        return {"status": "not_found", "detail": f"未找到手机号 {phone} 的搜索结果"}
    
    # ... 后续逻辑 ...
```

#### 5. 批量添加时的处理

```python
def add_contacts_batch(self, phones, verify_msg=None, interval=2):
    results = []
    for phone in phones:
        result = self.add_contact_by_phone(phone, verify_msg)
        
        # 根据状态分类统计
        if result["status"] == "phone_not_exist":
            logger.warning(f"手机号 {phone} 不存在，跳过")
        elif result["status"] == "already_friend":
            logger.info(f"手机号 {phone} 已是好友")
        elif result["status"] == "success":
            logger.info(f"手机号 {phone} 添加成功")
        
        results.append(result)
        time.sleep(interval)
    
    # 返回统计信息
    summary = {
        "total": len(phones),
        "success": sum(1 for r in results if r["status"] == "success"),
        "already_friend": sum(1 for r in results if r["status"] == "already_friend"),
        "phone_not_exist": sum(1 for r in results if r["status"] == "phone_not_exist"),
        "failed": sum(1 for r in results if r["status"] == "failed"),
        "details": results
    }
    return summary
```

在 demo 中添加状态处理示例：

```python
def add_single():

    else:
        print(f"❌ 添加失败: {result['detail']}")
```

## 其他改进

### 1. 增加超时重试机制

```python
def add_contact_by_phone(self, phone, verify_msg=None, max_retries=2):
    for attempt in range(max_retries):
        result = self._try_add_contact(phone, verify_msg)
        if result["status"] in ["success", "already_friend"]:
            return result
        if result["status"] == "phone_not_exist":
            return result  # 不存在不需要重试
        logger.warning(f"第{attempt+1}次尝试失败，重试...")
        time.sleep(2)
    return result
```

### 2. 保存失败截图便于调试

```python
if result["status"] in ["failed", "not_found"]:
    timestamp = time.strftime("%Y%m%d_%H%M%S")
    self.screenshot(save_path=f"fail_{phone}_{timestamp}.png")
```

### 3. 支持从文件读取手机号列表

```python
def add_contacts_from_file(self, filepath, verify_msg=None):
    with open(filepath, 'r', encoding='utf-8') as f:
        phones = [line.strip() for line in f if line.strip()]
    return self.add_contacts_batch(phones, verify_msg)
```
